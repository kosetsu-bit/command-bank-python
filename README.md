# command-bank-python
This is a simple banking console application written in Python. It has a few simple functions, such as money transfers, account replenishment, withdrawals, etc.

When you first launch the bank, the app asks for your first and last name, then asks if you want to deposit money into your account. After depositing money, you have six different actions to choose from. The app remembers your first and last name, bank balance, and deposit balance for only one session. While the session is active, you can interact with your balance as much as you want, but you can only top up your balance once when you log in to the app.

This is my first project, and the bank was built using AI (I wrote all the basic coding, occasionally correcting errors using AI, and also asking about new features and how to use them).
